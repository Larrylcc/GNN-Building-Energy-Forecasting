{% for step in gru.model.flow_steps %}
步骤 {{ loop.index }}

{{ step }}

{% if loop.index == 1 %}

训练脚本先按建筑、仪表和时间排序，确保序列窗口只来自同一条连续小时轨迹。

{% elif loop.index == 2 %}

标准化参数只由训练集估计，之后复用于 valid/test，避免评估集信息泄漏。

{% elif loop.index == 3 %}

若连续片段长度不足 168 小时，或片段中出现非 1 小时间隔，该位置不会生成训练窗口。

{% elif loop.index == 4 %}

nn.GRU 输出最后一层隐状态，隐状态可以视为前 168 小时历史的压缩表示。

{% elif loop.index == 5 %}

线性层只输出一个连续值，对应下一小时变换后目标尺度的读数。

{% elif loop.index == 6 %}

auto-rolling 更接近真实多步预测：后续窗口会看到模型自己的历史预测，而不总是真实历史读数。

{% endif %}
{% endfor %}
