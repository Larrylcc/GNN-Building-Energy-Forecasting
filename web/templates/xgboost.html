Structure

{% for step in xgboost.model.flow_steps %}
步骤 {{ loop.index }}

{{ step }}

{% if loop.index == 1 %}

三个文件已经在预处理阶段完成异常筛选、目标变换和时间顺序切分，训练时直接按 train / valid / test 三个集合读取。

{% elif loop.index == 2 %}

timestamp 用于切分和排序，meter_reading 是监督学习标签；剩余 34 列组成每条样本的特征向量。

{% elif loop.index == 3 %}

QuantileDMatrix 是 XGBoost 面向直方图训练的高效数据容器，会把连续特征按分位数近似压缩成分箱表示，从而降低内存占用并加快分裂点搜索。

{% elif loop.index == 4 %}

回归树是一组“特征条件判断”的树结构，每个叶子节点输出一个连续数值；XGBoost 每一轮新增一棵树，学习当前预测误差的梯度方向。

{% elif loop.index == 5 %}

最终预测不是单棵树给出，而是多棵回归树的输出累加，再得到变换后目标尺度上的 y_scaled。

{% elif loop.index == 6 %}

valid 用于训练过程中的早停和模型选择，test 用于最终泛化效果评估；两者都在同一变换后目标尺度上计算指标。

{% endif %}
{% endfor %}
